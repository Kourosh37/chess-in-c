UI font loading order:
1) assets/fonts/ui_font.ttf
2) assets/fonts/Cinzel-Bold.ttf
3) assets/fonts/PlayfairDisplay-Bold.ttf
4) system bold serif fallbacks (if available)
5) assets/fonts/NotoSans-Regular.ttf
6) default system fallback

To set a custom font for the whole UI, place your font file as:
assets/fonts/ui_font.ttf

Recommended for chess-style UI:
- Cinzel Bold
- Playfair Display Bold
